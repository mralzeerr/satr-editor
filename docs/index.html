لماذا سطر وليس الأدوات الأجنبية؟

🇸🇦

عربي أولًا، لا ترجمة لاحقة

واجهة RTL حقيقية، شرح للأخطاء بالعربية، ووكيل يفهم طلبك بلغتك — صُمم عربيًا من أول سطر.

🤖

وكيل يبني ويصلح ذاتيًا

لا يقترح أكوادًا فقط — ينشئ الملفات، يشغّل الأوامر، يقرأ الأخطاء من الطرفية ويصلحها بنفسه حتى يعمل تطبيقك.

🎙️

تكلّم بلهجتك

إدخال صوتي يفهم اللهجات الخليجية والعربية — قل ما تريد بناءه وشاهده يتشكّل أمامك.

🎓

وضع التعلّم

الوكيل يشرح لك ماذا فعل ولماذا، خطوة بخطوة بالعربية — أداة تعلّم برمجة بلا حاجز اللغة.

🧰

قوالب عربية أصيلة

ابدأ من قوالب RTL جاهزة مصممة بذوق عربي — لا قوالب أجنبية مترجمة.

🔓

مجاني ومفتوح المصدر

الكود كله أمامك على GitHub برخصة MIT، مفتاح API خاص بك وبياناتك على جهازك، وتحديثات تلقائية مع كل إصدار.

ابدأ في ثلاث خطوات

1

حمّل وثبّت

حمّل أحدث إصدار لويندوز — مثبّت عادي أو نسخة محمولة بلا تثبيت.

2

أدخل مفتاحك

احصل على مفتاح Anthropic API من console.anthropic.com وأدخله عند أول تشغيل — يُحفظ مشفرًا على جهازك ولا يغادره.

3

اطلب بالعربية

اكتب أو تكلّم: «أبغى متجرًا إلكترونيًا» — وشاهد الوكيل ينشئ الملفات ويشغّل المعاينة ويصلح الأخطاء بنفسه.

⚠️ يظهر لك تحذير Windows SmartScreen؟ هذا طبيعي ومؤقت

التحذير يظهر لأن الإصدارات الحالية غير موقّعة رقميًا بعد (شهادة التوقيع قيد الإجراء)، وليس لأن في التطبيق ضررًا — الكود كله مفتوح المصدر ويمكنك مراجعته بنفسك. للمتابعة:

  1. اضغط More info — مزيد من المعلومات
  2. ثم اضغط Run anyway — التشغيل على أي حال

فور اكتمال شهادة التوقيع الرقمي سيختفي هذا التحذير نهائيًا في الإصدارات القادمة — وستصلك تلقائيًا.
